# 🧠 Kin Keeper

Kin Keeper is a simple and efficient relationship management application that helps you keep track of your connections, interactions, and important people in your life.

---

## 🚀 Features

- 👥 Add and manage contacts
- 🏷️ Tag people (e.g., friend, family, colleague)
- 📅 Track last interaction time
- ⏰ Get reminders to reconnect
- 📝 Store notes and personal details
- 🔍 Search and filter contacts

---

## 🛠️ Tech Stack

- Next.js 15 (App Router)
- Frontend: React JS
- Styling: Tailwind CSS + DaisyUI
- Recharts
- Lucide React Icons
